# 🌸 Mummy's Mother's Day Surprise

A beautiful, interactive Mother's Day website made with love by Sakshi.

## 🚀 Deploy on GitHub Pages (Free)

Follow the steps below — no coding knowledge needed!

### Step 1 — Create a GitHub account
Go to https://github.com and sign up (free)

### Step 2 — Create a new repository
1. Click the **+** icon → **New repository**
2. Name it exactly: `mummy-surprise`
3. Set it to **Public**
4. Click **Create repository**

### Step 3 — Upload files
1. Click **uploading an existing file** on the repo page
2. Drag ALL the files from this ZIP (extracted) into the uploader
3. Make sure to include the hidden `.github` folder too!
4. Click **Commit changes**

### Step 4 — Enable GitHub Pages
1. Go to your repo → **Settings** → **Pages**
2. Under **Source** → select **GitHub Actions**
3. Click **Save**

### Step 5 — Wait 2 minutes ⏳
GitHub will automatically build and deploy your site.
Your live URL will be:
👉 `https://YOUR-USERNAME.github.io/mummy-surprise/`

## ✏️ How to Customize

Open `src/App.jsx` and look for these at the very top:

| Variable | What it changes |
|---|---|
| `YOUR_NAME` | Your name (currently "Sakshi") |
| `MOM_NAME` | What you call your mother (currently "Mummy") |
| `BALLOON_MESSAGES` | The 5 love messages in balloons |
| `LETTER` | The heartfelt letter text |
